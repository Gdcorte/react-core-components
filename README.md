# Core Components using React

This project is intended to provide react components for use in other projects.

This library was becoming quite large and components of the most diverse nature. Hence they were grouped and broken into different libraries and live in the following repos:

- [Theme](https://github.com/Gdcorte/react-core-theme) ([react-core-theme](https://www.npmjs.com/package/@gdcorte/react-core-theme))
- [Icons](https://github.com/Gdcorte/react-core-icons) ([react-core-icons](https://www.npmjs.com/package/@gdcorte/react-core-icons))
- [Menu](https://github.com/Gdcorte/react-core-menu) ([react-core-menu](https://www.npmjs.com/package/@gdcorte/react-core-menu))
- [Input](https://github.com/Gdcorte/react-core-input) ([react-core-input](https://www.npmjs.com/package/@gdcorte/react-core-input))
- [Button](https://github.com/Gdcorte/react-core-button) ([react-core-button](https://www.npmjs.com/package/@gdcorte/react-core-button))
- [DateTime](https://github.com/Gdcorte/react-core-date-time) ([react-core-date-time](https://www.npmjs.com/package/@gdcorte/react-core-date-time))

## How to test this library locally

For local development of this library, run `pnpm install` to install all the development dependncies.
Then you can run storybook with `pnpm dev` and see all of the components more in depth.
